# Analyse-des-performances-et-satisfaction-des-transports-urbains

## Présentation du projet
Ce projet analyse les données historiques et en temps réel de fréquentation des transports à Chicago et à Philadelphie afin d'évaluer la performance des transports et d'appuyer les décisions stratégiques.

## Technologies utilisées
- Python (pandas, numpy, scipy, matplotlib)

- Power BI (Power Query, DAX)

- API (CTA, SEPTA)

## Sources de données

- Chicago : RDF, Excel

- Philadelphie : CSV

- API en temps réel

## Structure du projet
- /notebooks : EDA et nettoyage
- /data : Données brutes et traitées
- /Dashboard : Fichier Power BI
- /data clean : Fichier CSV nettoyée

## Modèle de données
Schéma en étoile avec tables de faits (Itinéraire, Mode) et tables de dimensions .


## Indicateurs clés de performance (KPI)

- Nombre total de passagers
- Croissance mensuelle
- Comparaison des modes de transport

## Procédure d'exécution

1. Lancer les notebooks Python

2. Ouvrir le fichier Power BI

3. Actualiser les données
